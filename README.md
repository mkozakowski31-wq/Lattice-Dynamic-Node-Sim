# marko

Currently the PY script accesses the obj file then plots boundery vertices.
Goals:
- Make OBJ access computer agnostic (ie. not requiring a pathway)
- Automatically split edges to front, back, left side and right side
- create UV map using geodesic parameterization

Potential problems for you:
- libs may not be installed
- The pathway to OBJ is on my computer so the program may not be able to access the OBJ on your computer
